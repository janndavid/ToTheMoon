To The Moon

To The Moon is a Lua powered fullstack framework. Work is however in progress, and many features are still missing. As of now the template engine is provided by tarantool template. A custom template engine is however being coded as you read. Other features consist of a custom ORM (object relational mapper) which connects To The Moon with a sqlite database. Support for MySQL is also being worked on. The setup of the project can be found in the documentation.

The Tour

This website showcases a few of To The Moon's features. The built in login page can be found in the top bar by pressing the button. But to use the login feature you must first sign up.
